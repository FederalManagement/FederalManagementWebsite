== Description ==

Google+ Sharings Widget is a Wordpress plugin that shows your latest sharings on your Google+ profile or page on your wordpress blog.

== Requirements ==

In order to make this plugin works you need to:

* Know your Google+ Page or Profile Id (ie: 108916466184564256687 is my Id)
* Create a Google+ API key here : https://code.google.com/apis/console/

== Installation ==

If you don't use the Wordpress automated plugin installation, just :

1. unzip the archive in your wordpress plugin directory
2. go to your admin panel to the extension section
3. activate the plugin

== Frequently Asked Questions ==

How does the plugin work?

If everything went fine during the installation process, you should be able to see a new widget available in your widgets page. Just drag and drop the widget anywhere you want it to appear and fill in the form fields to customize it!
You're all done!
